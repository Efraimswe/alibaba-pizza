# Alibaba Kebab — Design Foundation

## Направление
**Neo-Brutalism × Vibrant Block-based** — «стрит-фуд постер / стикер-бук».
Референс-логика (ui-ux-pro-max): Vibrant & Block-based (bold, block layout, geometric,
high contrast) + Neo Brutalism Mobile (толстые рамки, hard offset shadows без blur,
лёгкие повороты карточек, стикеры). Подходит кебабной: энергия уличной еды,
ощущение физической доски меню, не выглядит шаблоном.

## Эмоция
Юзер заходит → «О, это что, постер? Прикольно» → голод → палец сам тянется к кнопке звонка.

## Палитра (Design-фаза; wireframe — только серые)
- `--brand-primary: #EA580C` (аппетитный оранжевый) / on-primary `#FFFFFF`
- `--brand-secondary: #F97316`
- `--brand-accent: #2563EB` (CTA-контраст, «доверие»)
- `--brand-bg: #FFF7ED` (тёплый крем) / fg `#0F172A`
- `--brand-yellow: #FFD93D` (стикеры-акции)
- Правило: оранжевый не для мелкого текста; контраст пар — AA 4.5:1.

## Типографика
- Display: **Space Grotesk** 700 (заголовки, цифры-индексы, цены)
- Body: **Karla** 400/500/700
- Цены и часы — tabular-nums.

## Форма
- Рамки 3–4px сплошные, радиус 0 (бейджи — 999).
- Тень = hard offset 4–6px, без blur (дёшево для GPU).
- Повороты карточек/стикеров: -2°…2°, не больше.
- Секции разделены «рваным»/зубчатым краем (CSS clip-path zigzag).

## Лейаут (mobile-first, 375px)
1. Sticky header: wordmark-блок ALIBABA + тогл NL/EN/FR (плейсхолдер).
2. Тикер-строка статуса: «OPEN VANDAAG 16:00–23:30 •» (потом marquee).
3. Hero: стопка гигантских слов PIZZA / PITA / GRILL блоками со сдвигом,
   стикер акции дня (повёрнут), адрес, CTA «BEL NU» + «MENU».
4. Меню = «чеки-тикеты»: каждая категория — нативный `<details>` с зубчатым низом,
   гигантский индекс 01–13, сочное фото категории 80px, счётчик позиций,
   «vanaf €N». Внутри — позиции БЕЗ фото: имя + ингредиенты + цена через
   dotted leader; klein/groot — две колонки.

5. Акции: 3 карточки-стикера (di/wo/do) с чередующимся поворотом.
6. Инфо: часы (таблица 7 дней, сегодня выделен), адрес + карта-плейсхолдер +
   «— m van jou» (geolocation в следующих фазах), телефон крупно.
7. Аллергены: строка-нота (полные 14 аллергенов ЕС — когда клиент даст данные).
8. Footer: плейсхолдеры Privacy/Cookies (req 3), копирайт.
9. Sticky bottom call-bar: «BEL 014 / 41 40 47», safe-area inset.

## Фото-слоты (`.wire-photo` — штриховка-фолбэк, рендер через next/image)
- Hero: 16:9 «signatuur schotel» — главный аппетит-кадр.
- Тикет категории: 80px квадрат в summary (одно сочное фото на категорию).
- У позиций меню фото НЕТ (решение руководителя 2026-07-14).
- Карточки акций: квадрат на всю ширину карточки.
- Источник: генерация pollinations + rembg (scripts/); клиентские фото заменят при наличии.

## Motion-таргеты (СЛЕДУЮЩАЯ фаза, сейчас НЕ делать)
- Тикер — бесконечный marquee (transform, pausable).
- `<details>` — плавное раскрытие тикета; «отрыв чека» при открытии.
- Стикеры — mechanical press (translate = offset тени) на тап.
- Стаггер входа позиций меню 30–50ms; всё transform/opacity, ≤300ms, reduced-motion.

## Wireframe-инварианты
- Только серая шкала (#fff → #171717), никаких брендовых цветов.
- Ноль анимаций и transition.
- Тап-таргеты ≥44px, контент не прячется под sticky-барами.
