# UI/UX Audit — Alibaba Kebab (2026-07-16)

Аудит по скиллу `uiux-audit-efraim` (20 правил школы). Вьюпорты: mobile 375×812 (основной), desktop 1440×900. Страницы: `/`, `/en`, `/fr`, `/privacy`, `/cookies` + интерактивные состояния (тикет меню, дропдаун языка, фокус с клавиатуры, клик-вне).

## Контекст
- Продукт: сайт пиццерии-кебабной, заказ только по телефону (afhalen).
- Главное действие: **звонок 014 / 41 40 47**.
- Целевая эмоция (из design-foundation.md): «О, это что, постер? Прикольно» → голод → палец тянется к кнопке звонка.

## Вердикт коротко
Блокеров (S1) нет: конверсионный путь чист — sticky bel-bar с `tel:` всегда на экране, эмоция голода на мобиле работает (фото, тёплая палитра). Главные долги: **десктоп-раскладка** (полэкрана пустоты), **дропдаун языка не закрывается**, **«сегодня» не выделен в часах** и пачка полировки.

## Что уже хорошо — НЕ трогать
- Конверсионный путь: sticky «BEL & BESTEL» с корректным `tel:+3214414047` (4 места), телефон в тикере и футере.
- Эмоция аппетита на мобиле: реальные фото блюд, тёплая палитра по foundation (#E23D28/#1E3B31/#FAF6EF), сочные фото категорий.
- Бенто-ремесло: радиусы консистентны (card 24 / inset 20 / pill full), разделение заливкой без бордеров, мягкие тени.
- Типографика: Lobster только в wordmark, Space Grotesk на заголовках/ценах, dotted leaders + tabular-nums в позициях меню — дорого.
- Доступность: видимый focus-ring (проверено Tab'ом), тап-зоны ≥44px (`min-h-11`/`min-h-16`), reduced-motion выключатель, аллергены по EU 1169/2011.
- Микровзаимодействия: marquee-тикер, press-scale, «+»→«×» морф, wiggle трубки.

## План фиксов (по приоритету)

### Фикс 1 · Десктоп-раскладка бенто (S2, находка 1)
**Где:** `components/landing.tsx` (секции Menu, Hier & nu), контейнер-классы `lg:`.
**Что:** на `lg:` двухколоночный бенто: тикеты меню слева (~60%), справа sticky-колонка «Hier & nu» (часы + карта + телефон крупно). Хиро: убрать вертикальные провалы — eyebrow/wordmark/sub единым ритмом (шкала 8pt), фото остаётся справа.
**Промпт агенту:** «Шкала отступов 4/8/16/24/32, внутри группы мало — между группами ×2. На lg: две колонки бенто: меню слева, sticky-инфо справа. Прищур-тест: одна доминанта, структура читается. Мобильную вёрстку не трогать.»

### Фикс 2 · Дропдаун языка (S2, находка 2)
**Где:** `components/landing.tsx:90-150`.
**Что:** закрытие по Esc, клику вне и после выбора языка (маленький client-обработчик поверх `<details>` или Popover API). Поведение — как у всех дропдаунов в вебе (правило 4: стандарты).
**Промпт агенту:** «Любое всплывающее закрывается по Esc, клику вне и после выбора. Сохрани текущую pop-in анимацию 180ms и focus-ring.»

### Фикс 3 · «Hier & nu» живой (S2, находка 3)
**Где:** `landing.tsx:417-430` + тикер.
**Что:** выделить текущий день (красная точка/плашка primary, день жирным), добавить статус-бейдж «Nu open · sluit 23:30» / «Opent om 16:00» над таблицей. Зебру убрать — она конфликтует с выделением сегодня (одно выделение на таблицу). Тикер может брать тот же статус.
**Промпт агенту:** «Эвристика №1: статус виден без чтения таблицы. Один акцент на таблицу — сегодня; зебру убрать. Часы уже tabular-nums — сохранить.»

### Фикс 4 · `<html lang>` по локали (S2, находка 4)
**Где:** `app/layout.tsx` (+ как локаль пробрасывается в `/en`, `/fr`).
**Что:** SSR должен отдавать `lang="nl-BE" | "en" | "fr"` по маршруту — убрать hydration mismatch (консоль сейчас в ошибках), это же SEO/скринридеры.

### Фикс 5 · Пакет полировки (S3, находки 5–9)
1. Плюрализация: «1 keuze» — `lib/i18n.ts` (для en/fr тоже: choice/choices, choix).
2. Неразрывный пробел в ценах: `€ ` в `eur()`-выводе и «vanaf €».
3. Футер: строку «Ondernemingsnummer:» скрывать, пока номера нет.
4. Кнопка «Sluit ×» в конце раскрытого тикета (скроллит к summary и закрывает).
5. Desktop: sticky call-bar показывать только после скролла мимо hero-CTA (или на десктопе — компактный вариант с номером).
**Промпт агенту:** «"Дорого" = тысяча маленьких совпадений: пройди по списку и почини каждое расхождение, ничего не редизайня.»

### Фикс 6 · Print/screenshot guard (S3, находка 10)
`@media print { .view-reveal { opacity: 1 !important; animation: none } }` — карточки не должны исчезать в печати/PDF/скриншотах.

## Не делать (осознанные решения руководителя — не «чинить»)
- Гигантский Lobster-wordmark в хиро вместо продуктового заголовка — это постер-бренд из foundation («О, это что, постер?»). Если когда-то захочется A/B: вариант с headline-обещанием еды + wordmark меньше.
- «vanaf €» в тикетах категорий — норм для категории из 39 позиций, внутри цены финальные.
- Фото у позиций меню отсутствуют — решение руководителя от 2026-07-14.
